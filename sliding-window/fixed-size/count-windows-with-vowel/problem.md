# Count Windows With At Least One Vowel

## Pattern
Fixed-Size Sliding Window

---

## Problem

Given a lowercase string `S` and an integer `k`,  
return the number of substrings of `S` with length `k`  
that contain **at least one vowel**.

Vowels are: `a, e, i, o, u`.

---

## Example

**Input**

S = "abcde"  
k = 2  

**Output**

2  

---

## Constraints

- 1 ≤ k ≤ S.length
- S consists of lowercase English letters only.

---

## Expected Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
