# Maximum Distinct Characters in Window

## Pattern
Fixed-Size Sliding Window

---

## Problem

Given a string `S` and an integer `k`,  
return the maximum number of **distinct characters**  
in any substring of `S` with length `k`.

---

## Example

**Input**

S = "aababcabc"  
k = 3  

**Output**

3  

---

## Constraints

- 1 ≤ k ≤ S.length

---

## Expected Complexity

- Time Complexity: O(n)
- Space Complexity: O(k)
