# Maximum Sum of k Consecutive Digits

## Pattern
Fixed-Size Sliding Window

---

## Problem

Given a string `S` consisting only of digits (`0–9`)  
and an integer `k`,  
return the maximum sum of digits among all substrings of length `k`.

---

## Example

**Input**

S = "12345123"  
k = 3  

**Output**

12  

---

## Constraints

- 1 ≤ k ≤ S.length
- S consists only of digits.

---

## Expected Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
