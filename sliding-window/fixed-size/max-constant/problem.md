# Maximum Consonants

## Pattern
Fixed-Size Sliding Window

---

## Problem

Given a lowercase string `S` and an integer `k`,  
return the maximum number of **consonants** in any substring of `S` with length `k`.

Vowels are: `a, e, i, o, u`.

---

## Example

**Input**

S = "abciiidef"  
k = 3  

**Output**

2  

---

## Constraints

- 1 ≤ k ≤ S.length
- S consists of lowercase English letters only.

---

## Expected Complexity

- Time Complexity: O(n)
- Space Complexity: O(1)
